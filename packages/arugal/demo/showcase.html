Testing Instructions

1. Start the Demo

pnpm run demo

This will build the library and start a local server at http://127.0.0.1:8080/demo

2. Basic Navigation Test

  • Scroll through the page to see all gallery types
  • Click any image to open the lightbox
  • Use arrow keys (‚Üê ‚Üí) to navigate in lightbox
  • Press ESC or click backdrop to close
  • Try clicking pagination dots
  • Use inline carousel arrows

3. Autoplay Test

  • Observe the second gallery advancing automatically
  • Hover your mouse over it - autoplay should pause
  • Move mouse away - autoplay should resume

4. API Test

  • Scroll to the "API Testing" section
  • Click each button and observe the status box
  • Try "Destroy Gallery" then "Reinitialize"

5. Mobile Test

  • Open DevTools (F12) and toggle device toolbar
  • Select a mobile device preset
  • Try touch gestures (swipe left/right)
  • Verify responsive layout

Code Examples

Basic Initialization

import { init } from '@ev-core/arugal';

const manager = init();

With Custom Options

import { init } from '@ev-core/arugal';

const manager = init({
    theme: {
        primaryColor: '#ff6b6b',
        backgroundColor: '#ffffff',
        textColor: '#333333'
    },
    inline: {
        autoplay: {
            interval: 3000,
            pauseOnHover: true
        }
    }
});

Create Single Gallery

import { create } from '@ev-core/arugal';

const element = document.querySelector('#my-gallery');
const gallery = create(element, {
    inline: {
        enabled: true,
        navigation: true
    }
});
